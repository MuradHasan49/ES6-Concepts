# 🚀 Modern JavaScript (ES6+) Concepts Practice

এই রিপোজিটরিটিতে আমি জাভাস্ক্রিপ্টের আধুনিক ফিচারগুলো (ES6+) প্র্যাকটিস করছি। বিশেষ করে অবজেক্ট ডিস্ট্রাকচারিং, অ্যারো ফাংশন এবং স্প্রেড অপারেটর নিয়ে কাজ করা হয়েছে।

## 📋 প্রজেক্টের উদ্দেশ্য (Objectives)
- ES6 এর নতুন সিনট্যাক্সগুলো আয়ত্ত করা।
- নেস্টেড অবজেক্ট থেকে সহজে ডাটা বের করা (Destructuring)।
- ক্লিন এবং রিডেবল কোড লেখার অভ্যাস করা।
